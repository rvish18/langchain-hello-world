# LangChain Hello World

A simple demonstration project using LangChain to build AI-powered applications.

## Features

- Basic LangChain setup
- Example usage of LangChain components

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/rvish18/langchain-hello-world.git
   cd langchain-hello-world
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
   Or if using pyproject.toml:
   ```bash
   pip install -e .
   ```

## Usage

Run the main script:
```bash
python main.py
```

## Requirements

- Python 3.8+
- LangChain
- Other dependencies as listed in pyproject.toml
